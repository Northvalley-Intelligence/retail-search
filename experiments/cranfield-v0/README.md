# Cranfield v0 Experiment

This experiment records Phase 1 baseline artifacts.

Current generated artifacts:

- `cranfield-sample-bulk.ndjson`: sample bulk indexing payload generated by `npm run prepare:index`
- `evaluation-sample.json`: fixture metric output generated by `npm run validate`
- `evaluation-live.json`: live OpenSearch evaluation artifact generated after loading the full public Cranfield corpus

Fixture artifacts validate shape and math only. Phase 1 live acceptance is backed by the real `cranfield-v0` OpenSearch index and the public Cloudflare Worker at `https://retail-search.feroshjacob.workers.dev`.
