Fixture artifacts validate shape and math only. Phase 1 live acceptance is backed by the real `cranfield-v0` OpenSearch index and the public Cloudflare Worker at `https://retail-search.feroshjacob.workers.dev`.

`SE-0002` keeps the public baseline unchanged while testing lexical candidate architecture behavior. The best lexical Cranfield candidate is refined `prf-rerank`, but it remains transferability-pending. See `docs/evaluation/cranfield-failure-groups.md`, `docs/evaluation/cranfield-prf-rerank-research.md`, and `docs/evaluation/timeline.md`.

GEN-017 adds the fast-loop workflow: refresh first-stage retrieval pools from remote OpenSearch only when needed, then run reranker tuning locally from `retrieval-pools-field-sum-top50-gen017.json`.

GEN-018 adds vector/hybrid infrastructure. The local-hash control run validates the mechanics but remains below refined PRF.

GEN-019 adds a real local Ollama embedding run. Ollama `llama3.1:8b` embeddings slightly improved cached field-sum retrieval but remained below refined PRF, so `ARCH-0.3-candidate` is still not promoted.

GEN-020 adds an offline LTR path. Cross-validated LTR improves field-sum but remains below refined PRF; in-sample LTR exceeds refined PRF, so this is useful ranking-capacity evidence but not promotion evidence. The Ollama embedding features did not help the cross-validated ablation.

GEN-021 adds boosted-tree LTR. The stronger depth-3 run improves graded cross-validation to 0.3216 nDCG@10 but still remains below refined PRF, while binary cross-validation remains below the GEN-020 lexical LTR ablation.

GEN-022 adds Hugging Face BGE retrieval embeddings. BGE hybrid retrieval becomes the first embedding evidence above refined PRF, with best graded nDCG@10 0.3533 and best binary nDCG@20 0.4915. BGE LTR features also beat refined PRF under query-grouped cross-validation, with boosted-tree graded nDCG@10 0.3603.

GEN-023 validates BGE hybrid retrieval in remote OpenSearch. The separate candidate index `cranfield-v0-bge-base-en-v15-gen023` contains all 1,400 documents plus 768-dimensional BGE vectors. Remote hybrid retrieval reaches graded nDCG@10 0.3533 and binary nDCG@20 0.4926. The public/default architecture remains unchanged until transferability and promotion policy are addressed.
